# Listing Copy Style — Duckwerks

Reference for all eBay (and Reverb) listing copy. Edit this file between sessions to iterate on results.

---

## Titles

- 80-character max on eBay
- Lead with the most searchable specs (brand, model, series, issue numbers, key attributes)
- No punctuation padding — every character should earn its place
- No em-dashes in titles

**Examples:**
```
ElfQuest Hidden Years #1-25 27 29 + Holiday Special 28-Issue Lot Warp Graphics
Boss CE-2 Chorus Pedal MIJ Japan Vintage 1980s
TEAC A-2300SX Reel to Reel Tape Deck Serviced Working
```

---

## Description Structure

1. **Story/intro paragraph** — what is this, why does it matter, what makes it worth buying. 2-4 sentences. First-person, conversational. No specs here.
2. **Specs block** — short labeled lines. Format: `Label | Value`. No sentences.
3. **Free shipping paragraph** — verbatim, every time (see below)
4. **"Sold As Is."** — on its own line, always last

---

## Free Shipping Paragraph (Verbatim)

Use this exact text on every listing:

```
All my listings ship with free shipping for your ease, none of this $30 shipping on a 1 pound item. I price my listings fairly but please feel free to make an offer.
```

For comic lots, append this sentence:
```
Comics are packed securely with cardboard backing to protect against bending in transit.
```

---

## Closing Line

Always end with this on its own line:
```
Sold As Is.
```

---

## Local Pickup Variant

When listing for local pickup (FB Marketplace, Craigslist, or eBay local):
- Replace the free shipping paragraph with:
```
Located in the Sunset District, San Francisco. Local delivery within 30 minutes (no bridges). Bay Area delivery available for a fee.
```

---

## Condition Field (eBay)

The condition field is separate from the description. Use it for:
- Factual summary of what's included (issue numbers, completeness)
- Condition assessment in plain language
- Any notable flaws or bonuses

Keep it factual and specific. No story here — that's what the description is for.

---

## Style Rules (Non-Negotiable)

- **No em-dashes** anywhere — not in titles, descriptions, condition fields, or any other output. Use a comma, a period, or rewrite the sentence. Hyphens in compound words (high-end, near-complete) are fine.
- **No double-hyphens** as em-dash substitutes
- **Round numbers only** — never .99 pricing. Round to nearest $5 or $10 depending on price tier
- **No bold or markdown formatting** in eBay description text — plain text only
- **Specs as labeled lines**, not bullet points or prose
- **Free shipping baked into price** — never charge shipping separately on eBay listings

---

## Pricing Tiers for Rounding

| Price range | Round to |
|---|---|
| Under $20 | Nearest $1 |
| $20–$100 | Nearest $5 |
| $100+ | Nearest $10 |

## Category-Specific Notes

### Comics / Graphic Novels
- Always state condition using standard grades (VG, VF, NM, etc.) plus plain-language description
- Note bagged/boarded status in both description and condition field
- For lots: state what's included, what's missing, and duplicate count
- Shipping method: Media Mail

### Music Gear
- Note country of manufacture if visible (MIJ, MIT, USA)
- Note whether original case, power supply, cables are included
- For vintage gear: note any mods, service history, or original components
- Platform note: Reverb typically outperforms eBay 20-40% on music gear

### Disc Golf
- Note plastic type, weight, color, and condition (unthrown/mint/field worn)
- Note stamp edition if limited or notable
- Condition claim "unthrown" or "mint" is a meaningful premium signal — only use if accurate

### Consumer Electronics
- Note generation/revision — do not comp or list across versions
- Note PSU/power supply inclusion explicitly
- Note any known cosmetic issues
